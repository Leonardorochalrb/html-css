O que é semântica?

num sistema linguístico, o componente do sentido das palavras e da interpretação das sentenças e dos enunciados, e é o foco do HTML.
E para tudo que é forma, existe o CSS3.

Semântica nas tags

existem tags com a mesma função, e por isso o estudo de semântica é crucial. Um bom exemplo é: <u>
Essa linha está sendo escrita utilizando a tag <u>

Se eu quero me referir a algum endereço, e preciso deixar destacado, a tag <u> não vai ter semântica, e sim, estilo. Um endereço, para ter semântica, deve ser escrito, por exemplo, desta forma:

Rua flamengo, limoeiro, 229


Percebe que o endereço ficou em fonte Italico? Isso não quer dizer que a tag <address> deixa os textos em Italico, tudo é questão de semântica.

O que realmente significam HTML5 e CSS3?

HTML5 é a versão mais recente da linguagem de marcação de hipertexto HTML, e diferente de seu antecessor, o HTML4, o HTML5 tem seu foco completo na semântica dos elementos HTML.

Isso é CSS3 e não HTML5!

Principais formatações

Negrito

Existem duas formas de deixar uma palavra ou frase em negrito: a não semântica, usando a tag <b>. Essa é a forma não semântica. E a forma semântica é utilizando a tag <strong>. Essa é a forma semântica.

Ênfase

As frases, ou termos em destaque, em ênfase, funcionam da mesma maneira. A forma não semântica é utilizando a tag <i> para ênfase.
A forma certa é utilizando a tag <em> para ênfase.

Texto Marcado

Use a tag mark para um marcador de texto

Texto Excluído

Podemos marcar um texto como excluído, que pode ser lido, mas não considerado.

Texto Inserido

Podemos marcar um texto como inserido, para dar uma ênfase, e indicar que ele foi adicionado depois.
Existe também a tag U, que tem a mesma formatação. Mas não é semântica.
